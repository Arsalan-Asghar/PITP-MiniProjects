## Repository Structure

The repository is divided into the following directories to keep the projects organized:

* 📁 **`Data-Analysis/`** - Scripts and folders dedicated to cleaning, processing, and analyzing datasets.
* 📁 **`Management-System/`** - Logic-based applications designed to handle records, inventory, and administrative tasks.
* 📁 **`Utility-Tools/`** - Practical, everyday scripts built to help with personal organization and planning.

## Projects Overview

Here is a breakdown of the specific projects contained within each folder:

### 📊 Data-Analysis
* **`Sales Data Cleaning/`**: A sub-directory containing scripts focused on preprocessing and formatting raw sales data.
* **`Sales Data Analysis.py`**: A tool to analyze cleaned sales data, identify trends, and calculate key metrics.
* **`Movie Ratings Analysis.py`**: An analytical script to process, aggregate, and draw insights from movie rating datasets.

### 🏢 Management-System
* **`Inventory Management System.py`**: A program to track inventory levels, add new items, update quantities, and manage stock efficiently.
* **`School Management System.py`**: A system designed to manage student records, track attendance, or handle general school administrative tasks.

### 🛠️ Utility-Tools
* **`Shopping List Budget Planner.py`**: A handy application to help users create shopping lists while tracking estimated costs against a set budget.
* **`To Do List Manager.py`**: A task management script allowing users to create, update, delete, and mark daily tasks as complete.
